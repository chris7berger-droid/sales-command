   - QBCallbackPage.jsx: handles OAuth redirect from Intuit

   - "Connect QuickBooks" button on Invoices page header
   - Shows "QB Connected" indicator once linked
   - Tokens stored in qb_connection table (Supabase)

2. Proposal → Sold = Create Customer/Job in QB
   - qb-create-job Edge Function
   - Creates parent customer (company) if not found
   - Creates sub-customer (job) under parent
   - Sub-customer format: "{jobNum} {CO} - {CustomerName} - {JobName}"
   - Skips job name if same as customer name (prevents duplication)
   - Maps jobsite address to billing/shipping
   - Maps contact info (email, phone) from customer record
   - Stores QB sub-customer ID on call_log.qb_customer_id
   - Stores QB parent ID on customers.qb_customer_id
   - Triggered from Internal Approve AND customer signing flows
   - Non-blocking — runs in background

3. Invoice → Sent = Create Invoice in QB
   - qb-sync-invoice Edge Function
   - Uses "Services" item for all line items (not cost codes)
   - Work type name used as line item description
   - Maps location from jobsite_state to QB Department
   - Maps invoice #, date, due date, discount, description/PO#
   - Stores QB invoice ID on invoices.qb_invoice_id
   - Triggered when invoice is sent via Stripe
   - Non-blocking

4. Invoice → Paid = Record Payment in QB
   - qb-record-payment Edge Function
   - Creates Payment linked to QB Invoice
   - Handles net amount (after discount)
   - Triggered from manual "Mark as Paid" button
   - Triggered from Stripe webhook (checkout.session.completed)
   - Non-blocking
