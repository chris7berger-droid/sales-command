WHAT WAS DONE
===============================================================================

1. Fixed invoice delete not removing invoices from the list
   - File: src/pages/Invoices.jsx
   - Root cause: handleDelete() and handleVoidConfirm() used hard DELETE,
     which RLS silently blocked. The load() query also had no deleted_at
     filter, so even soft-deleted rows would appear.
   - Fix: Both delete paths now set deleted_at instead of hard-deleting.
     The load() query now filters with .is("deleted_at", null).
   - Invoice lines are left intact (belong to the soft-deleted invoice).
