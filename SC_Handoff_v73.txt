2. BUILT ImportToLiveWizard
   New file: src/components/HistoryLocker/ImportToLiveWizard.jsx (~615 lines)

   7-step wizard (modeled on NewInquiryWizard):
     1. Customer Type (Commercial / Residential)
     2. Match or Create — case-insensitive name match against customers;
        auto-selects match or prefills new-customer form with archive name
     3. Contact & Billing — billing name + email REQUIRED; billing_terms
        defaults from tenant_config.default_billing_terms (30 fallback)
     4. Addresses — jobsite prefilled from raw_data address/*; business
        address blank (Glide export never has it; user types from contract)
     5. Sales Rep — auto-match team_members.email ←→ raw_data
        "job/salesPerson_email"; picker fallback
     6. Work Types — fuzzy match against raw_data "job/Type of Work"
        (exact → includes → token-overlap); picker allows override
     7. Review — grid summary + Import button

   raw_data keys used for prefill (all "/" paths — Glide export shape):
     - customer/customerName, customer/ifCustomerName, customer/Customer Name
     - customer/Phone number
     - job/Project Name
     - job/Type of Work
     - job/Bid Due Date (parsed via new Date())
     - job/salesPerson_email
     - job/soldAmount, job/proposalAmount (fallback)
     - address/Address, address/City, address/State, address/Zip

   On Import (transactional-ish, no explicit tx):
     a. customers insert (only if customerMode = "new")
     b. call_log insert: stage="Sold", archive_record_id set,
        job_number auto-incremented (max+1)
     c. job_work_types insert (selected work type ids)
     d. proposals insert: status="Sold", total=parseMoney(soldAmount),
        proposal_number=1, archive_record_id set, signing_token = random UUID

   After save: onSaved({ proposalId, jobId }) — parent navigates to
   proposals page with the new proposal opened.

3. WIRED "IMPORT TO LIVE" BUTTON ON ArchiveRecordDetail
   Top-right teal button, shown only when:
     - canImport (Admin or Manager), AND
     - record.record_type === "call_log"
   Opens the wizard as a modal overlay.

   Prop thread:
     App.jsx
       → Archive (userRole, onNavigateProposal)
         → ArchiveSearchView (canImport, onNavigateProposal)
           → ArchiveRecordDetail (canImport, onNavigateProposal)
             → ImportToLiveWizard (onSaved)

   App.jsx onNavigateProposal sets initialProposal={openId:id} and

4. NO SEPARATE "View attachments from archive" LINK
