39a8be1 — fix(qb): dedupe "X - X" job_name pattern before pushing to QB DisplayName
  supabase/functions/qb-create-job/index.ts gets a new dedupRepeated()
  helper near the top of the file. Splits on " - "; if segment count

  existing showJobName logic + subName builder both see the cleaned
  value. No DB changes, no new dependencies. Catches "X - X" and
  "A - B - A - B" patterns; ignores partial overlaps and odd segment
  counts on purpose. Closes nothing in BACKLOG — this is a preventive
  fix for a finding raised mid-session, not a previously-filed item.
