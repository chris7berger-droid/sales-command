   `escapeHtml(input: string)` lied — runtime accepted INT and
   crashed on `.replace()`.

   fn invoke (ProposalPDFModal.jsx:73-83). When the fn 500'd, DB still
   showed Sent. The UI lied. F28 fixes this architecturally.
3. No post-deploy smoke ritual for edge fns. Deploy exit 0 was
   treated as success. 4 errored invocations sat in logs before user
   repro. F29 + the new global memory rule address this.
4. H11 was a 138-LOC bundled refactor (auth + body shape + escaping
   + signing URL + allowlist). Bug anywhere = silent failure
   everywhere. Future security refactors should split into ≤50-LOC
   commits, each smoke-tested independently.
