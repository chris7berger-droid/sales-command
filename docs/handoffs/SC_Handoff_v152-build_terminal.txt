=== What this feature does ===
Proposals whose pricing era (pricing_anchor_at ?? created_at) >= noon Central 2026-06-26 bill to the
EXACT penny (Math.round to cent). Pre-cutoff proposals keep the legacy round-UP (Math.ceil). Safe
default = ceil everywhere (missing/unparseable era, wrong object, thin embed -> ceil), so no unwired
path silently under-bills. A multi-GC clone inherits its SOURCE's era via pricing_anchor_at, so a
clone made post-cutoff from a pre-cutoff source still bills ceil (freeze == bill).
