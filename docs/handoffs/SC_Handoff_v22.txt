WTC CALCULATOR ARCHITECTURE (key facts)
====================================================

- WTCCalculator.jsx lives in src/pages/ NOT src/components/
- CustomerSigningPage is built inside WTCCalculator.jsx (internal preview only)
- PDFPreviewModal is built inside WTCCalculator.jsx
- PublicSigningPage.jsx is the customer-facing signing page (separate file)
- autosave: debounced 1.5s, upsert pattern
- work_type_id is INTEGER (1-40), not UUID
- WTC has 7 tabs: Bidding Info, Labor, Materials, Scope of Work, Travel, Discount, Summary
- Summary tab: Lock & Approve → Generate PDF
- On autosave: writes proposalPrice back to proposals.total

====================================================
