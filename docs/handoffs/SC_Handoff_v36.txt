EDGE FUNCTIONS
====================================================

qb-auth (--no-verify-jwt):
  - action: "exchange" — OAuth code → tokens
  - action: "refresh" — refresh expired access token
  - action: "status" — check if QB is connected

qb-create-job (--no-verify-jwt):
  - Creates parent customer + sub-customer (job) in QB
  - Input: { callLogId }
  - Sub-customer format: "{jobNum} - {JobName}"

qb-sync-invoice (--no-verify-jwt):
  - Creates invoice in QB under the job's sub-customer
  - Uses "Services" item, work type name as description
  - Input: { invoiceId }

qb-record-payment (--no-verify-jwt):
  - Creates payment in QB linked to QB invoice
  - Input: { invoiceId }
