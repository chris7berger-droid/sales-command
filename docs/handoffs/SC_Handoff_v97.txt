PR #15 (92b0696) — fix/job-number-uniqueness
  Partial unique index on
    call_log (tenant_id, job_number, COALESCE(co_number, 0))
    WHERE archived = false AND job_number IS NOT NULL
  COALESCE(co_number, 0) lets parent (NULL co_number) coexist with
  COs (1..N) under same job_number, while rejecting two distinct
  parents sharing a job_number per tenant. Wizard retry guards on
  PG 23505 in NewInquiryWizard (auto-allocate retry; override path
  surfaces clear error) and ImportToLiveWizard (legacy-num path
  surfaces clear error; auto path retries).
  Migration: 20260507130000_call_log_unique_job_number.sql
            (applied to prod cleanly — pre-flight cleanup confirmed
             no active dupes remained).
  Files: +69 / -31
